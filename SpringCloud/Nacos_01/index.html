【Nacos】01-Nacos的安装

发表于 2021-11-15 更新于 2021-11-16 阅读次数： Changyan： 本文字数： 2.6k 阅读时长 ≈ 2 分钟

Nacos的安装

下载

  1. 页面地址：https://github.com/alibaba/nacos

  2. 我这里使用https://github.com/alibaba/nacos/releases/tag/1.4.2版本

安装

Nacos至少有三台才能组成集群，因此需要有三个Nacos服务

  1. 解压并分复制出三份

    tar -zxvf nacos-server-1.4.2.tar.gz -C /usr/local/

    cp -r nacos/ nacos-cluster/

    cd nacos-cluster

    cp -r nacos/ nacos1/

    cp -r nacos/ nacos2/

    mv nacos/ nacos3/

  2. 修改配置文件-连接MySQL数据库(application.properties)

    vi application.properties

  3. 执行SQL文件

    sql文件路径/usr/local/nacos/conf/nacos-mysql.sql

  4. 修改集群配置文件-配置集群(cluster.conf)

    cp cluster.conf.example cluster.conf

    hostname -i

    vi cluster.conf

  5. 修改启动执行文件

    vi startup.sh

  6. 启动

    /usr/local/nacos-cluster/nacos1/bin/startup.sh -m cluster

    /usr/local/nacos-cluster/nacos2/bin/startup.sh -m cluster

    /usr/local/nacos-cluster/nacos3/bin/startup.sh -m cluster

    我的服务器2G内存，启动一个Nacos已经顶不住了，第二个直接启动失败。

启动

单机

startup.sh -m standalone

集群

startup.sh -m cluster

Nacos页面

浏览器输入http://ip:8848/nacos进入登录页。账号密码默认均为nacos

登陆成功后
